SHIX is an enterprise-grade AI-powered IT Service Management (ITSM) platform designed as an intelligent overlay for Freshservice.

## Purpose
SHIX assists IT teams by:
- Analyzing tickets using AI
- Retrieving accurate knowledge via RAG
- Proposing and applying resolutions
- Maintaining full auditability and security

## Architecture Overview
- **Frontend**: React + Vite + TypeScript
- **Backend**: FastAPI (Python)
- **AI Layer**: RAG, GraphRAG, Agentic AI
- **Infrastructure**: Ubuntu VPS, systemd, Nginx, SSL

## Key Features
- AI-assisted ticket resolution
- Secure writeback to Freshservice
- Audit logging
- Policy-aware AI responses
- Enterprise-ready design
